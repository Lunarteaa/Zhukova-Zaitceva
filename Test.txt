1.	Что делает команда git status?
А  Показывает состояние проекта
Б  Показывает имя и email пользователя 
В  Показывает место занимаемое репозиторием на жёстком диске

2. Что делает команда git add?
A  Создает файл с указанным именем и сразу выгружает его в Git
Б  Начинает отслеживать указанный файл
В  Синоним команды git commit

3. Что означает статус файла untrecked?
А. Что файл был удален
Б. Что система не отслеживает файл
В. Что файл находится не в репозитоии git

4. Что означает статус файла new?
А. Что файл только что был создан и не отслеживается
Б. Такого статуса не существует 
В. Что файл только начал отслеживаться и не имеет истории.

5.  Что означает статус файла modified?
А Такого статуса не существует
Б Что файл имеет историю в системе и был недавно изменен 
В Означает что файл был добавлен в коммит

6.  Когда надо использовать команду git status?
А Как можно чаще 
Б Всегда при создании коммита
В Только для того чтобы узнать статус файла 

7.  Что делает команда git log?
А Показывает историю коммитов
Б Используется для входа в систему
В Синоним команды git status

8.  Какой командой установить глобальный email?
А $ git config --global user.email 
Б $ git config --global .email 
В $ git config --global user. johndoe@example.com

9.  Какой командой можно просмотреть все установленные настройки ?
А $ git config --global user 
Б $ git config --list --show-origin
В $ git config --list --show

10.  Что делает команда git init ?
А Создаёт в текущем каталоге новый подкаталог с именем
Б Создает новый каталог
В Удаляет файл в текущем каталоге
